### EVIC ###

EVIC is a fork and rewrite of [erlffmpeg](https://github.com/emedia-project/erlffmpeg).


### Video ###

For video conversion, EVIC use ffmpeg.


### Image ###

For image conversion, EVIC use ImageMagick.
